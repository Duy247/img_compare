# Windows

No need to install any external dependencies.

Add dlls folder to path.\

Then install Python dependencies:

```bash
pip install opencv-python cairosvg
```

# Mac OS 

Install pkg-config and cairo:

```bash
brew install pkg-config cairo
```

Then install Python dependencies:

```bash
pip install opencv-python cairosvg
```
